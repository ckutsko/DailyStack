Use it like an app on iPhone

  • Open this site in Safari.
  • Tap Share, then Add to Home Screen.
  • Open the new Home Screen icon.
  • Tap Enable reminders and allow notifications.
  • Keep the app open briefly so daily reminders can be scheduled again if needed.

Note: iPhone web apps support notifications on modern iOS, but scheduled reminders can be affected by browser and system limits. For mission-critical medication reminders, Apple Reminders remains the most reliable fallback.
